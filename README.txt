English:
To play the game launch ...\Risk\bin\Risk.bat .
Maps are saved in ...\Risk\bin\MapMaker as .json files.

Deutsch:
Um das Spiel zu starten launcht man ...\Risk\bin\Risk.bat .
Karten werden in ...\Risk\bin\MapMaker als .json Datei gespeichert.

Die zwei wahrscheinlichsten Probleme wenn das Spiel nicht richtig launcht sind:
- Es ist Java installiert bzw. die Java Version ist älter als 24
- Es ist Java 24 oder neuer installiert aber die JAVA-HOME Variable ist nicht gesetzt
Um beide einfach zu beheben empfehle ich das Temurin JDK zu installieren 
Link: https://adoptium.net/de/temurin/releases/?os=any&arch=any&version=24
Im install-wizard muss dann die JAVA-HOME setzen option ausgewählt werden.
