English:

General Controls:
  - All ingame menus/dialogues can be exited by pressing <BACK>
  - Full Menus (Settings, PLAY etc.) can be exited by pressing the -MAIN MENU- button

Camera Controls:
- <UP> <DOWN> <LEFT> <RIGHT> move the camera in their respective direction
- <ZOOM IN> <ZOOM OUT> controls the zoom of the camera
- <RIGHT CLICK> + <MOUSE DRAG> moves the camera

Gameplay Controls:
- <BACK> opens the pause/mission menu
- <LEFT CLICK> adds/removes clicked country to selection if possible
- <UNDO> clears the selection 
- <ACCEPT> initiates an action
  - Enter the number ~n~ you want to perform the action with
  - The performed Action is decided by the game phase
    - !If the selection is not valid the action will not be performed!
    - Startup: Place ~n~ troops
    - Setup: Place ~n~ troops
    - Attack: Attack with ~n~ troops or reinforce conquered Country with ~n~ troops
    - Fortification: Move ~n~ troops
- <SHIFT> + <ACCEPT> or <CYCLE> starts the next game phase if possible
- <SHIFT> + <UNDO> undoes the last game action if possible
- <FUNCTION 1> opens the continent legend
- <FUNCTION 2> opens the card menu
  - If you do not have any cards the menu will show "No Cards"
  - Select cards in the list and press <ACCEPT> to add/remove them to the selection
    - If the cards in the selection can be traded in the -Trade Cards- button will appear

Map Designer Controls:
- <BACK> opens the pause menu
- <SHIFT> + <CYCLE> cycles the Map Designer mode
- Polygon Editing:
  - <LEFT CLICK> adds a vertex to the current polygon at the mouse position
  - <SHIFT> + <LEFT CLICK> selects the clicked polygon
  - <UNDO> undoes the last vertex of the current polygon
  - <ACCEPT> finalizes the polygon if possible
  - <FUNCTION 1> toggles the visibility of the polygon body
  - <FUNCTION 2> toggles the visibility of the polygon outline
- Edit Connections:
  - <LEFT CLICK> adds/removes the clicked country to the selection
  - <SHIFT> + <UNDO> clears the selection
  - <ACCEPT> toggles the connection between the selected countries if the selection is valid
  - <SHIFT> + <ACCEPT> creates automatic connections
  - <SHIFT> + <LEFT CLICK> + <MOUSE DRAG> creates a cosmetic line from the start position to the end position of the mouse
  - <CYCLE> cycles the selected cosmetic line
  - <UNDO> removes a cosmetic line if one is selected
- Select Country:
  - <LEFT CLICK> adds/removes the clicked country to the selection
  - <SHIFT> + <LEFT CLICK> opens the country naming dialogue for the clicked country
  - <UNDO> removes the last country added to the selection
  - <ACCEPT> opens the continent selection list if there are any continents and the selection is valid
    - Adds countries in the selection to the selected continent
  - <FUNCTION 1> or <FUNCTION 2> toggle the continent view


Deutsch:
 
Allgemeine Steuerung:
  - Alle Menüs/Eingabefenster in der Spielhandlung werden mit <BACK> geschlossen
  - Vollständige Menüs (Settings, PLAY etc.) können mit dem -MAIN MENU- Taste geschlossen werden

Kamera Steuerung:
- <UP> <DOWN> <LEFT> <RIGHT> bewegen die Kamera in der angegebenen Richtung
- <ZOOM IN> <ZOOM OUT> kontrollieren den Zoom der Kamera
- <RIGHT CLICK> + <MOUSE DRAG> bewegt die Kamera

Spiel Steuerung:
- <BACK> öffnet das Pause-/Mission-Menü
- <LEFT CLICK> fügt das geklickte Land der Auswahl hinzu bzw. entfernt diese aus der Auswahl
- <UNDO> löscht die Auswahl
- <ACCEPT> startet eine Aktion
  - Man tippt die Zahl ~n~ ein mit der die Aktion durchgeführt wird
  - Die Aktion ist von der Spielphase abhängig
    - !Falls die Auswahl inkorrekt is wird keine Aktion durchgeführt!
    - Startup: Platziert ~n~ Truppen
    - Setup: Platziert ~n~ Truppen
    - Attack: Angriff mit ~n~ Truppen or verstärkt erobertes Land mit ~n~ Truppen
    - Fortification: Bewege ~n~ Truppen
- <SHIFT> + <ACCEPT> oder <CYCLE> startet die nächste Spielphase falls dies möglich ist
- <SHIFT> + <UNDO> macht die letzte Aktion rückgängig falls möglich
- <FUNCTION 1> öffnet die Kontinent-Legende
- <FUNCTION 2> öffnet das Karten-Menü
  - Falls sie keine Karten habe wird "No Cards" angezeigt
  - Wählen sie eine Karte aus der Liste aus und fügen sie diese mit <ACCEPT> der Auswahl hinzu bzw. entfernt diese aus der Auswahl
    - Falls die Auswahl eingetauscht werden kann wird die -Trade Cards- Taste sichtbar

Map Designer Steuerung:
- <BACK> öffnet das Pause-Menü
- <SHIFT> + <CYCLE> durchläuft die Map Designer Modi
- Polygon Editing:
  - <LEFT CLICK> fügt dem ausgewählten Polygon einen Vertex auf bei der Mausposition hinzu
  - <SHIFT> + <LEFT CLICK> wählt den geklickten Polygon aus
  - <UNDO> macht den letzten Vertex des ausgewählten Polygons rückgängig
  - <ACCEPT> speichert den Polygon falls es möglich ist
  - <FUNCTION 1> schaltet die Sichtbarkeit des Polygon-Körpers um
  - <FUNCTION 2> schaltet die Sichtbarkeit des Umrisses des Polygon um
- Edit Connections:
  - <LEFT CLICK> fügt ein Land zur Auswahl hinzu bzw. entfernt ein Land
  - <SHIFT> + <UNDO> enfernt alles aus der Auswahl
  - <ACCEPT> schaltet die Verbindung zwischen zwei Ländern falls die Auswahl gültig ist
  - <SHIFT> + <ACCEPT> schafft automatisch Verbindungen
  - <SHIFT> + <LEFT CLICK> + <MOUSE DRAG> schafft eine kosmetische Linie vom Startpunkt zum Endpunkt des Mausbewegung
  - <CYCLE> durchläuft die Auswahl der kosmetischen Linie
  - <UNDO> entfernt die ausgewählte kosmetische Linie
- Select Country:
  - <LEFT CLICK> fügt das geklickte Land der Auswahl hinzu bzw. entfernt diese aus der Auswahl
  - <SHIFT> + <LEFT CLICK> öffnet das Eingabefenster zur Benennung des geklickten Landes
  - <UNDO> entfernt das zuletzt hinzugefügte Land aus der Auswahl
  - <ACCEPT> öffnet das Kontinent-Auswahlfenster falls es Kontinente gibt und die Auswahl nicht leer ist
    - Fügt die Länder in der Auswahl dem ausgewählten Kontinent hinzu
  - <FUNCTION 1> oder <FUNCTION 2> schaltet die Kontinent-Sichtweise um
